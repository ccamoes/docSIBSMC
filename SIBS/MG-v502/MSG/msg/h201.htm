(1) -  Na primeira consulta (primeira ocorrência da mensagem de pedido) são aceites valores a zeros. Nas consultas subsequentes, os campos devem ser preenchidos com os dados obtidos na última ocorrência devolvida na última mensagem de resposta.

(3) - Valores possíveis:   "="  Indica que se pretendem obter os elementos iguais aos dados indicados.
">"  Indica que se pretendem obter os elementos superiores aos dados indicados,   i.e., os valores seguintes.
"<"  Indica que se pretendem obter os elementos inferiores a partir dos dados indicados, i.e., os valores anteriores.
   
(4) - Valores possíveis:   01 a 04
