V02 -  Relativamente à V01, na mensagem S202 foi eliminado o campo 233 (CODMOEDA), o campo 1024 foi expandido, e o campo 1709 (APLIC_N) foi incluído no Header.


(1) - Valores possíveis:   "="  Indica que se pretendem obter os elementos iguais aos dados indicados.
">"  Indica que se pretendem obter os elementos superiores aos dados indicados,   i.e., os valores seguintes.
"<"  Indica que se pretendem obter os elementos inferiores a partir dos dados indicados, i.e., os valores anteriores.
 
(2) - Valores possíveis:   00  - Por Inicializar
02  - Em funcionamento
04  - Fora de Serviço
05  - Em Supervisão

(3) - São aceites valores a zeros (lista a partir da primeira data disponível).
