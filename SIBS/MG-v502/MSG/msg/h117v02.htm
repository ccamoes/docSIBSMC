V02 -  Relativamente à V01 e no âmbito da versão 5 do Modelo Global, esta mensagem foi alterada para permitir uma maior flexibilização da gestão dos dados dos acordos. 


(a) - Serão aceites valores a zeros (lista a partir do 1º elemento disponível).
 
(1) - Valores possíveis:   ">"  Indica que se pretendem obter os elementos superiores aos dados indicados.
"<"  Indica que se pretendem obter os elementos inferiores a partir dos dados indicados.
   
(2) - Valores possíveis:  01 a 16

(*)  A posição indicada é referente à 1ª ocorrência do registo.
