NOTA:

Esta operação só pode ser efectuada por um Terminal ou Host do Emissor do cartão.

(a)  Este campo indica se estão preenchidos os campos (b) ou (c).
Se = 1 estão preenchidos os campos (b) e os (c) estão a zeros.
Se = 2 estão preenchidos os campos (c) e os (b) estão a zeros.

A introdução da data de expiração só é mandatória para os Bancos cujo acesso à base de dados se realize de acordo com o cenário 3 - Número de cartão único composto por PAN e Data expiração. Ver Livro II, capítulo A.2.5.


(d)  Este campo é opcional, ou seja, pode ser zeros e não tem decimais. Caso esteja preenchido, o mínimo são 5 euros e o máximo são 500 mil euros.


(e) O código de moeda será 000 (zeros) se o montante for zero.


(f)  No caso das mensagens Host-to-Host, estes campos podem estar a zeros ou preenchidos como o Banco entender, de forma a que seja possível auditar a operação executada.
