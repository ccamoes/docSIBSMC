EXEMPLOS DE LISTAGEM PARA COMERCIANTE


A informação mensal com o resumo das operações de Fecho do Terminal é apresentada num dos seguintes tipos de listagem:

  • Listagem das Operações de Fecho
    Este tipo de listagem é emitida para o comerciante quando o tipo de serviço indicado pela Entidade de Apoio do Terminal, nos dados do acordo MB, é ´1`- Com Extracto por Fecho Local.
    Nesta situação é emitida uma listagem com os dados dos Fechos Contabilísticos de todos os TPAs do comerciante/estabelecimento.
  • Listagem das Operações Multibanco por Fecho de TPA/Depósito PMB
    Este tipo de listagem é emitido para o comerciante quando o tipo de serviço indicado pela Entidade de Apoio do Terminal, nos dados do acordo MB, é ´3`- Por Fecho mais Comissões.
    Nesta situação é emitida uma listagem por TPA do comerciante/estabelecimento.
  • Listagem das Operações de Fecho Segundo um Tipo de Cartão Aceite.
    Este tipo de listagem destina-se aos comerciantes que têm acordo com o Representante de uma dada Marca/Cartão, sendo a listagem emitida em nome do Representante. A emissão destas listagens pela SIBS depende de um contrato directo com o Representante da Marca.
