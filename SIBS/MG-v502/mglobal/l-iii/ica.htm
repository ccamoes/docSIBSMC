NOME FICHEIRO:  IMAGENS DE CHEQUES PARA ARQUIVO "FICH" = ICA

 

OBJECTIVO


O ficheiro ICA destina-se a disponibilizar ao Banco de Apoio do Terminal imagens de cheque para arquivo. Trata-se de um ficheiro com um formato de imagem (JPEG) que cumpre a regulamentação do Banco de Portugal para este efeito.

 

ESTRUTURA


O ficheiro é constituído por:

  • um registo de Header "TIPREG=0";
  • registos de Detalhe "TIPREG=1";
  • um registo de Trailer "TIPREG=9".


O comprimento deste ficheiro é variável porque depende do tamanho dos campos das imagens de cheques.
